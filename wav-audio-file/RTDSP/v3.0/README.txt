Real Time Audio Delay for GNU-Linux systems.
Version 3.0

This application supports .wav audio files, 16bit and 24bit.
Number of channels and sampling rate depends on your audio hardware.

Compilation notes: 
2 APIs must be explicitly linked: -lpthread and -lasound
ALSA API build resources are needed for this application.
These resources can be installed with package "libasound2-dev"
